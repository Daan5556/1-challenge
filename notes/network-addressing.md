## Addressing

The network layer is responsible to assign unique addresses to nodes in the
network. Most used one: `IP Address`.

An IP address consist of 32 bits. First 3 bytes is network, last one is host.

## Subnetting

Dividing a large IP network into smaller subnets.

Subnet communication is efficient and safe. Inter subnet communication is routed
trough a router.
